# Guardian Agent 交接说明（2026-07-15）

## 当前已完成

1. 起夜场景 Agent
   - 已支持起夜离床、去洗手间、喝水、头晕、求助、摔倒、无响应超时、返床确认等分支。
   - 已新增老人原话入口：`POST /api/v1/guardian/conversations/night-turn`。
   - 已新增 GuardianMessage 标准消息入口：`POST /api/v1/guardian/messages`。
   - 已新增场景模拟入口：`POST /api/v1/guardian/scenarios/{scenario_code}`。
   - 已支持外部文字或语音转写文本触发 Agent 决策时间线。

2. 健康日报和周报
   - 已有三位老人模拟健康数据。
   - 风险卡片切换后，日报、周报、最近日常数据会跟随对应老人切换。
   - MCP 工具已压缩为工作流方式，避免超过涂鸦平台工具数量上限。

3. 家庭回忆库
   - 已生成 30 段模拟通话录音和回忆数据。
   - 已支持按人物、情感、主题、回忆发生时间、录音记录时间检索。
   - 回忆卡片支持播放原录音片段。

4. 前端网页
   - 演示控制台按钮可触发起夜、SOS、健康异常、回忆生成等模拟流程。
   - 点击风险卡片后会自动切换到对应老人。
   - 已修复网页默认选中已关闭事件的问题。
   - 已新增网页自动刷新，每 3 秒拉取一次最新 Agent 状态。

5. 语音模型接入准备
   - 已新增桥接脚本：`integrations/guardian_voice_bridge.py`。
   - 文本模式已验证可用，例如：

   - 真实语音模式入口已留好：

## 队友接续时建议顺序

1. 解压交接包。
2. 进入项目目录后安装基础依赖：
   `python -m pip install -r requirements.txt`
3. 启动本地网页服务：
   `python .\server.py`
4. 打开：
   `http://127.0.0.1:8765`
5. 先点击网页里的“触发夜间离床”或“触发标准场景”生成风险卡片。
6. 再测试文本桥接：
   `python .\integrations\guardian_voice_bridge.py text "我去趟卫生间，不用担心" --elder-id E001`
7. 如要继续测试真实语音，需要准备 `voice_lab.py` 所在目录，并用 Python 3.11 启动 `guardian_voice_bridge.py voice`。
